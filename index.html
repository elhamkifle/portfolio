InfoMediCure
Infomedicure is a web application that I designed to streamline and enhance the healthcare experience for patients and medical professionals. This project helped me enhance my skills in full-stack web development, utilizing HTML, CSS, JavaScript, Python, MongoDB, and Node.js.

Key Features:
1. Event Hosting: Infomedicure allows users to host events related to healthcare and medical topics. Users can create, update, and delete events, providing a platform for knowledge sharing and community engagement.
2. Disease Information Creation: The application provides doctors with the ability to create and manage information on various diseases. This feature allows doctors to share their expertise and educate users about different medical conditions.
3. Full CRUD capabilities of features 1 and 2: The application supports full CRUD (Create, Read, Update, Delete) capabilities of the two features mentioned above, with various users having various and distinct privileges.
4. User Authentication and Authorization: Infomedicure includes a secure user authentication system, ensuring that only authorized individuals can access and modify the application's features.
5. Responsive Design: The application is designed to be responsive and accessible across different devices and screen sizes.

To Do List
TodoList is a feature-rich web application designed to help users manage their tasks and increase productivity. I created this project with a little twist from the usual todo lists by giving the users options to create either a personal or a work to do list.

Key Features:
1. Separate to do lists for work and personal use: with work to do lists having additional features like deadlines and an assignment feature to help allocate tasks to different people.
2. Data Persistence: The application utilizes local storage to store and retrieve task data. This ensures that users' task lists are saved and accessible across multiple sessions, providing a seamless user experience.
3. Responsive Design: The application is designed to be responsive and accessible across different devices and screen sizes.
